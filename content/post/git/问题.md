# 问题 1
## 问题描述
Failed to connect to 127.0.0.1 port 8889: Connection refused
## 解决步骤
查看代理使用情况
```
env | grep -i proxy
```
取消代理端口为 8889 的设置
```
unset HTTPS_PROXY
unset HTTP_PROXY
```
