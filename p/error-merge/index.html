消除 merge 操作

方法一

  1. 首先 git log 查看提交记录，找到出错的前一笔提交的 commit_id: 310f23a070195b57e245ccbfc67f8a6226154568
  2. 用命令 git rebase -i commit_id，查找提交记录
1

➜  git-test git:(branch-a) git rebase -i 310f23a070195b57e245ccbfc67f8a6226154568
