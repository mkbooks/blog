使用Python操作hive

在 Ubuntu20.04 系统下，使用 Python 操作 hive，这里使用 pyhive 包

Sep 16, 2022
阅读时长: 2 分钟

安装依赖包

1
2
3
4

pip install sasl
pip install thrift
pip install thrift-sasl
pip install PyHive

如果安装 sasl 出现错误，尝试以下解决方法

1

sudo apt-get install libsasl2-dev

代码样例

1
2
3
4
5
6
7
8

from pyhive import hive

conn = hive.Connection(host='192.168.122.251', port=10000, username='root', database='default', auth='NOSASL')
cursor = conn.cursor()
cursor.execute('SELECT * FROM test LIMIT 10')
for i in cursor.fetchall():
    print(i)
conn.close()
